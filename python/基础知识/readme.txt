数据类型
Python有五个标准的数据类型：

Numbers（数字）
    int (有符号整型)
    long (长整型[也可以代表八进制和十六进制])
    float (浮点型)
    complex (复数)

String（字符串）
    由数字，字母，下划线组成的一串字符

List（列表）
    完成大多数集合类的数据结构实现。支持字符，数字，字符串甚至可以包含列表(所谓嵌套)

Tuple（元组）
    类似List(列表), 使用()标识，内部元素使用逗号隔开。不能二次赋值，相当于只读列表
